# SignaRDotNet6Server
基于.Net 6.0的SignalR通讯Demo，包含WPF服务端，Web服务端，WPF客户端项目代码

注意：

1.WPF服务端不支持Win7及以下系统，会报Dll错误

2.目前不支持从服务端往客户端发送信息（已提交到SignalR Issue）
